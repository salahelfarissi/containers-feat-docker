# Docker

``` shell
# Check if docker is installed
$ docker version

# Build docker image
$ docker build -t docker-app .

# List docker images
$ docker images

# Run docker image
$ docker run docker-app

# Pull ubuntu image
$ docker run ubuntu

# List running processes
$ docker ps

# List running and non-running processes
$ docker ps -a

# Run ubuntu interactively
$ docker run -it ubuntu
```
